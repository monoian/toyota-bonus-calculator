請先閱讀這個 repository 的 README.md、AGENTS.md、docs/CODEX_HANDOFF.md、docs/ARCHITECTURE.md、index.html、assets/app.js、data/*.json。

這是一個要部署到 GitHub Pages 的 Toyota 台灣一線業務單筆成交獎金計算器。第一版請維持純前端，不要引入後端，也不要加入真實敏感獎金資料。

請先做架構檢查與重構建議：
1. 檢查目前檔案路徑、資料檔命名與 README 是否一致。
2. 找出計算邏輯與 DOM 操作耦合的地方。
3. 提出並實作最小重構，把計算邏輯抽成可測試的純函式。
4. 新增最小測試案例，涵蓋固定獎金、百分比獎金、階段獎金、累計獎金、支援金、保險乙式、影音與負調整。
5. 不要大改 UI；若要改手機版，先提出方案與理由。

完成後請說明修改了哪些檔案、如何測試、以及下一步建議。
